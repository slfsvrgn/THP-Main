Installation

What you need

  • Template download .zip from ThemeForest
  • FTP client
  1. To install this item you need your own server space. The server space can be purchased at many different hosting companies around the internet.
  2. After purchasing, you'll notice in your Downloads page of your ThemeForest admin area, that your item is there. Click download. You will receive a .zip file.
  3. Locate the .zip file on your computer and double click on it to open it.
  4. Within the .zip folder are a handful of folders and files. In the folder labelled “Documentation” are the help files for both the HTML and PSD templates. In the HTML folder is where your HTML template is located. You'll notice that theses files have extensions: .html, .css, .jpgs, and .js. The HTML files contain the core structure of the template, while the CSS files contain the template styles. The Javascript files (.js) are how the transitions and animations are added, and the Image (.jpg) files are image elements that are within the template. If you're a beginner, do not worry about looking within or editing the js files as they will look very foreign to you.
  5. Now you need to have a FTP client in order to access your server. There are many FTP client’s that you can download for free. Using the FTP client, you can now upload the all the files in the HTML folder onto your server location. Once these files have uploaded, you can open up a browser and type in your domain name to view.

Modification

What you need

  • HTML editor
  1. Now that your site is uploaded, it's time to put content in it. If you do not have a HTML editor on your computer then you are going to have to download one. Feel free to choose whatever software you like the best.
  2. Begin by opening your home-landing.html file into your editor. If you want this file to open when you type in your domain, rename it index.html. Here we will highlight HTML tags you'll need to look for.
  3. Title tag — This tag is going to give a title to your site, which will show up on browsers and search engines.
  4. Link tag — This tag is your links. You need to insert the URL of where you want to the link to go to within the "href" quotation marks.
  5. Image tag — This tag is for your images. If you look at your download folder, you'll see a folder called "images". All images that you want on your site should be uploaded into this folder. This template will resize your images to fit the dimensions of the image place holder. Once the image is uploaded into your images folder then go back to your HTML editor and place the image location URL between the "src" quotation marks. Make sure and use the correct extension: .png, .jpg, etc.
  6. H1, H2, H3, H4, H5 tags — These are different sizes for your headings. H1 will be the largest and H5 will be the smallest. The search engines will read these headings in order of importance, with H1 as the most important.
  7. P tag — This is where you will put all your textual content. The p tag stands for paragraph. Feel free to play around with the length of your content.
